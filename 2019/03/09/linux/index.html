linux的小白之路

2019 3.9

周末在家自己琢磨linux，刚开始无比顺畅，但是现在卡住了，先说一下流程。

win10系统需要下载ubuntu，虽然我不知道有什么用，然后根据网上的流程来看就是需要打开计算机配置里的linux的一个开关，可是我没找到，于是就先开始打开ubuntu了，下载了一个东西，接着就让我注册账户和密码，名字叫“zhouzhou”，（高能）填写完密码，回车后密码就消失了？？接着就显示我下载的一个东西好了，我就以为不需要密码了，没想到之后你所需要的所有的功能和命令都是需要密码的，关键是密码不仅不能输入我还不知道密码是啥，，，又看了半天发现，输入shell就可以输入数字了，但是我靠，，，，密码我不知道嗷嗷，连输入几次我都不正确，唉ε=(´ο｀*)))，卡在这样的一个尴尬的位置简直了，，，，下周还要继续接着搞，加油菜粥。

下午，我又把软件卸载，重新加载了一下，成功设置了密码，但是我还是不太懂怎么去使用和运行linux嗷嗷
